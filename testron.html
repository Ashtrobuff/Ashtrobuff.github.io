Testron is a fast, lightweight API testing app designed for mobile developers and testers who need to interact with REST APIs on the go.

üîß Features

  • Send GET, POST, PUT, and DELETE requests
  • Custom headers & request body support
  • Supports Basic Authentication
  • Handles JSON, XML, plain text, and even images
  • Auto-saves request history for easy reuse
  • If the URL returns an image (e.g. PNG/JPG), it's displayed and can be saved to your photo library

üì• How to Use

  1. Open the app and enter your API endpoint.
  2. Select the method: GET, POST, PUT, or DELETE.
  3. Optionally, add headers and a request body.
  4. For secured APIs, toggle Basic Auth and provide your username/password.
  5. Tap Send to view the response.
  6. If the response is an image, tap it to save it to your device.
